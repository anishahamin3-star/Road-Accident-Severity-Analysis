About the Analyzer System

The Road Accident Hotspot Prediction and Severity Analysis System is a professional data-driven dashboard designed to identify regional accident hotspots and analyze severity variables on Indian roadways.

Backend Framework: Flask (Python)
Data Processing: Pandas
Geographical Mapping: Folium Leaflet Heatmaps & Clusters
Visual Analytics: Headless Matplotlib Plotting
ML Severity Prediction: Rule-based heuristic weighting

By identifying hotspots and correlating severities with weather conditions, road type, and time of day, this system enables transportation agencies to allocate safety infrastructure proactively.

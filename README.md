# 🚦 Road Accident Severity Analysis and Prediction System

## 📌 Overview
This project is a Machine Learning-based Road Accident Severity Analysis and Prediction System. It analyzes accident data, identifies accident-prone areas, and predicts accident severity using machine learning techniques.

## ✨ Features
- Data preprocessing and cleaning
- Accident data analysis
- Hotspot detection
- Accident severity prediction
- Interactive graphs and visualizations
- User-friendly dashboard

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit
- Visual Studio Code

## 📂 Project Structure
- Source Code
- Dataset
- Machine Learning Model
- Dashboard
- Visualizations

## 🎯 Outcome
The system helps analyze accident patterns and predict accident severity, which can assist in improving road safety and decision-making.
